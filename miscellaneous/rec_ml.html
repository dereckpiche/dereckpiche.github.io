Machine Learning Recommendations

General

The combination of Probabilistic Machine Learning (Advanced Topics) by Kevin Murphy and Probabilistic Machine Learning (Introduction) by Kevin Murphy forms the only good modern textbook in the field, which is still too young to have a shorter summary. Understanding Deep Learning contains some nice theoretical exercises and the style is immaculate. Certainly the prettiest ML textbook so far. The Deep Learning Book was the first good Deep Learning textbook, although you should read Murphy's now as it is clearer and more up to date. Colah's essays bring nice blog-post style explanations and visualisations. However, the best visualizations are, as usual, given by 3B1B in his Neural Networks series. <

Calculus and Linear Algebra

These are by far the two most important branches of mathematics for scientists and engineers. Start with Vector Calculus, Linear Algebra, and Differential Forms: A Unified Approach, my favourite mathematics textbooks. Even contains some analysis for the curious. Then you might wanna read Linear Algebra Done Right by Sheldon Axler to get a little training in abstraction. If you are still hungy for abstraction, use Mathematical Analysis for Machine Learning looks okay. Multivariable Calculus by Stewart is also a gold standard.

Probabilities

Start with The Art of Probability by Richard Hamming. Information Theory, Inference and Learning Algorithms is by far the most interesting textbook on information theory. Simon Lacoste-Julien's Notes for IFT6269 given at MILA are great to understand both advanced probability theory and algorithms.

Reinforcement Learning

Start with Reinforcement Learning: An Overview by Kevin Murphy or Spinning Up RL from OpenAI. There is also Pierre-Luc Bacon's Reinforcement Learning Book, which is amazingly precise for some concepts. For the game theory in MARL I suggest Introduction to Game Theory by Tadelis.
